# Patient Monitoring System

A web-based Patient Monitoring System built using Python (Flask) and MySQL.  
It helps healthcare staff manage patient records, track health data, and view insights through a simple dashboard.

# Features
-  Login & Signup (secure authentication)
-  Add new patients
-  View all patient records
-  Update patient details
-  Patient health graph
-  Health suggestions
-  Quick health check

# Tech Stack
- Python (Flask)
- HTML, CSS
- MySQL
- Pandas, Matplotlib

# How to Run
1. Install required libraries:
   pip install flask mysql-connector-python pandas matplotlib

2. Run the app:
   python app.py

3. Open browser:
   http://localhost:5001
